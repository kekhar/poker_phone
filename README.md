## Кратко о проекте

**Poker Phone** это мобильное приложение для домашних покерных партий без сложной настройки и без отдельного сервера.

Один игрок создаёт стол и становится хостом, остальные находят его в локальной сети или подключаются по QR-коду. Приложение помогает быстро собрать лобби, занять места за столом и начать партию в формате **Texas Hold'em** для **2-6 игроков**.

## Как запустить

### Что понадобится

- [Flutter SDK](https://docs.flutter.dev/get-started/install)

## Что это за приложение

**Poker Phone** создан как удобный цифровой стол для игры в покер с друзьями в одной локальной сети. Это не онлайн-казино и не клиент для удалённой игры через интернет. Идея приложения в том, чтобы превратить обычные смартфоны в быстрый и понятный интерфейс для домашней партии.

После первого запуска игрок задаёт имя, выбирает аватар или ставит своё фото. Дальше приложение предлагает три основных сценария: создать лобби, подключиться к уже созданному столу или открыть тренировочный стол для знакомства с интерфейсом.

Хост создаёт игровое лобби и делится подключением с другими участниками. Для входа не нужно вручную настраивать серверную часть: приложение умеет искать столы в локальной сети, а также поддерживает подключение по QR-коду. Это делает старт партии заметно проще, особенно когда игроки сидят рядом и используют один Wi-Fi.

Отдельный акцент сделан на удобстве живой игры. Интерфейс ориентирован на быстрый вход, понятную навигацию, визуально аккуратную посадку игроков за стол и отображение состояния раздачи. Приложение подходит для дружеских игр, тестов механик и домашних покерных вечеров, где хочется убрать лишнюю организационную возню и сосредоточиться на самой партии.

## Основные возможности

- Создание локального лобби для игры в **Texas Hold'em**
- Подключение игроков через **QR-код**
- Автоматический поиск доступных столов в **LAN / Wi-Fi**
- Поддержка от **2 до 6 игроков**
- Настройка имени и аватара игрока
- Возможность использовать собственное фото профиля
- Тренировочный стол для проверки интерфейса и сценариев игры
- Экран стола с визуальным представлением игроков и текущего состояния партии
- Локальный формат без обязательного подключения к интернету
